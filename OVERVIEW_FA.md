# برنامه کلاسی هفتگی - نمای کلی پروژه

## 🎨 طراحی جدید

### رنگ‌بندی
این برنامه با الهام از لوگوی انجمن علمی هوش مصنوعی دانشگاه سیستان و بلوچستان طراحی شده است:

- **طلایی اصلی**: #C9A961
- **طلایی روشن**: #E0C589
- **آبی نفتی**: #1B3A52
- **آبی تیره**: #0F2537

### ویژگی‌های بصری
- طراحی مبتنی بر کارت (Card-Based Design)
- افکت‌های Glassmorphism
- نقوش سنتی بلوچی در پس‌زمینه
- انیمیشن‌های نرم و حرفه‌ای
- فونت فارسی Vazirmatn

## 📱 صفحات و بخش‌های اصلی

### 1. هدر برنامه
```
┌─────────────────────────────────────────────────────┐
│  🎓 لوگو  |  برنامه کلاسی هفتگی  |  👤 نام دانشجو │
│                                     🌙 تم   ✏️ ویرایش│
└─────────────────────────────────────────────────────┘
```

**عناصر**:
- لوگوی انجمن علمی
- عنوان برنامه
- نمایش اطلاعات دانشجو
- دکمه تغییر تم (روشن/تاریک)
- دکمه ویرایش اطلاعات

### 2. داشبورد آمار
```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   تعداد    │  │   ساعات    │  │  کلاس‌های  │
│   کلاس‌ها  │  │    کل      │  │    امروز   │
│     12     │  │     24     │  │      3     │
└─────────────┘  └─────────────┘  └─────────────┘
```

**اطلاعات نمایش داده شده**:
- تعداد کل کلاس‌ها
- مجموع ساعات درسی
- تعداد کلاس‌های امروز

### 3. انتخابگر روز
```
┌───────────────────────────────────────────────────┐
│ [همه] [شنبه] [یک‌شنبه] [دوشنبه] [سه‌شنبه] [چهارشنبه] │
└───────────────────────────────────────────────────┘
```

**قابلیت‌ها**:
- فیلتر بر اساس روز
- نمایش تعداد کلاس‌ها در هر روز
- قابل سوایپ در موبایل

### 4. نمایش کارت‌های کلاس
```
┌─────────────────────────────────┐
│  📚 ریاضی پیشرفته         ✏️ 🗑️│
│  👨‍🏫 دکتر احمدی                │
│  ─────────────────────────────  │
│  📅 شنبه | ⏰ 08:00 - 10:00    │
│  🏫 کلاس: ۲۰۱                  │
│  📝 امتحان: ۱۴۰۳/۰۹/۱۵        │
└─────────────────────────────────┘
```

**هر کارت شامل**:
- نام درس
- نام استاد
- روز و ساعت
- شماره کلاس
- تاریخ امتحان
- رنگ دسته‌بندی
- دکمه‌های ویرایش و حذف

### 5. دکمه‌های عملیاتی
```
┌──────────────────────────────────────────┐
│  [+ افزودن کلاس]                        │
│  [📄 دانلود PDF] [👁 پیشنمایش PDF]      │
│  [📊 دانلود Excel] [🔗 اشتراک‌گذاری]    │
│  [🗑️ پاک کردن همه]                      │
└──────────────────────────────────────────┘
```

## 🎯 جریان کاری کاربر

```
1. ورود به برنامه
   ↓
2. وارد کردن اطلاعات دانشجو (اولین بار)
   ↓
3. افزودن کلاس‌ها
   ↓
4. مشاهده برنامه هفتگی
   ↓
5. فیلتر/جستجو در کلاس‌ها
   ↓
6. دانلود PDF/Excel یا اشتراک‌گذاری
```

## 💡 نکات کلیدی

1. **ذخیره‌سازی محلی**: تمام داده‌ها در localStorage مرورگر ذخیره می‌شوند
2. **عدم نیاز به سرور**: برنامه به طور کامل سمت کاربر اجرا می‌شود
3. **واکنش‌گرا**: در تمام اندازه‌های صفحه نمایش به خوبی کار می‌کند
4. **قابل دسترس**: با توجه به استانداردهای WCAG طراحی شده
5. **امن**: تمام کتابخانه‌های خارجی با SRI بررسی می‌شوند
